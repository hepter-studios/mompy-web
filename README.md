# Mompy Website

Official landing page for Mompy.

Mompy is a retro learning console for Python programming. A focused desktop experience where learners practice real code through lessons, missions, progress, feedback, and challenges.

## Links

- Website: https://mompy.co
- App repository: https://github.com/hepter-studios/mompy
- Releases: https://github.com/hepter-studios/mompy/releases
- Discord: https://discord.gg/fqxvyGFyfa

## Local Preview

```powershell
python -m http.server 8781
```

Then open:

```text
http://127.0.0.1:8781
```
