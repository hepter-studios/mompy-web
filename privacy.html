Mompy Privacy Policy

Policy version: June 22, 2026

Mompy is a retro Python learning project designed for beginners. This policy explains, in plain language, how the Mompy website handles privacy and what happens when you use links to external services.

Mompy does not currently provide user accounts, payments, private profiles, or a hosted learning dashboard on this website. The website is primarily a public product page that links to resources such as GitHub, Discord, Python.org, and project downloads.

1. Information we collect

The Mompy website does not ask you to create an account or submit personal information through a contact form. If the hosting provider enables standard analytics or security logging, it may process basic technical data such as page requests, browser type, approximate region, device information, and performance information.

2. External services

Some Mompy links open third-party services, including GitHub, Discord, sponsor pages, Python.org, and release downloads. Those services operate under their own privacy policies and terms when you leave the Mompy website.

3. Downloads

Download actions point to official Mompy project releases or related project resources. The platform hosting those files may process standard request data when you download or view a release.

4. Community links

If you join the Mompy Discord, open a GitHub issue, or participate in GitHub Discussions, the information you post there is handled by the relevant platform. Do not share sensitive personal information in public community spaces.

5. Changes to this policy

This policy may be updated as Mompy grows, especially if new hosted features, accounts, analytics, or downloads are added. The version date at the top of the page shows when this document was last revised.

6. Contact

For privacy questions about Mompy, use the official project channels on GitHub.
